# 关于&致谢 <!-- {docsify-ignore-all} -->

作为一个在现代社会中生活的人类，自由访问互联网是每个人应有的最基本的权利。但是在某些国家或地区，这一基本权利却被剥夺了。这就迫使这些国家或地区的人们需要掌握突破互联网封锁的技术手段，因为只有这样才能最大限度的保证自己对于自由互联网的访问。

世界范围内真正的互联网不是洪水猛兽，它是世界上各个交流的人们彼此交流，互通有无的最佳媒介。对于互联网的封锁从教育方面来说是有百害而无一利的。从计算机方面来说最先进的技术都在美国以及欧洲众多国家，最精华的文章原文都是英文。这不是崇洋媚外，而是对自己和世界的真实的认识。由于互联网封锁的现状，太多的学生以及计算机从业者由于没有掌握访问自由互联网的技术手段，只能受困于质量惨不忍睹的中文局域网网络，这让人不能不联想到前朝"睿智"无比的愚民政策。自信的开放与讨论才是真正的文化自信，在"不敢说"、"不能看"、"不能听"的环境下提出的文化自信只能让人贻笑大方。

本书基于 Linux 内核提供一个学习路线。从网络编程到加密手段，一整套学习的路线下来，你可以掌握对于开发一个加密的、可自由访问互联网的服务器软件所需要的全部技术。关于本书存在的任何问题以及建议，均可以发送邮件，也可以直接在本页下方留言。
